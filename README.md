# Ficius Juris Club

Site web du **Ficius Juris Club** — club dédié à la formation, l'information et l'accompagnement dans le domaine du droit.

## Pages

- `index.html` — Page d'accueil
- `Formulaire.html` — Formulaire d'inscription

## Lancer le projet localement

```bash
python -m http.server 8080
```

Puis ouvrez [http://localhost:8080](http://localhost:8080) dans votre navigateur.
